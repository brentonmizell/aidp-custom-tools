# Testing — aidp_catalog_toolkit

Upload **aidp_catalog_toolkit.zip** in the workspace file browser, add a Custom Tool node, set the Tool Class Name, then use the Test panel with the values below. Mirror the Display Name / Description / Input Schema from the package's `tool_config.json`.

> Prerequisite: Needs AIDP Data Lake API + RP. Set region + data_lake_ocid (workspace_id for WorkspaceFile).

## CatalogFileTool
Read or list files in a Standard Catalog volume.

Config to set:
- region = us-ashburn-1
- data_lake_ocid = <OCID>

**Test 1: Read by names**

| Field | Value |
|-------|-------|
| `operation` | `get` |
| `catalog` | `construction_catalog` |
| `schema` | `construction_schema` |
| `volume` | `construction_documents` |
| `path` | `/safety_manual.md` |

Expected: file content.

**Test 2: List a volume**

| Field | Value |

| `catalog` | `construction_catalog` |
